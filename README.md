# 💖 Algoritmos e Estruturas de Dados (AEDs)

✨ Repositório dedicado aos estudos, implementações e projetos desenvolvidos na disciplina de Algoritmos e Estruturas de Dados — PUC Minas

---

## 🌸 Sobre a disciplina

AEDs é uma das bases mais importantes da Ciência da Computação (sim, aquela que faz você questionar suas escolhas às 2 da manhã).

O foco está no estudo de **estruturas de dados** e na forma correta de utilizá-las para resolver problemas de maneira eficiente.

Aqui você vai encontrar desde implementações simples até soluções mais elaboradas envolvendo organização e manipulação de dados.

---

## 📚 Conteúdos estudados

💡 Estruturas fundamentais:

* Listas (simples, duplamente encadeadas e circulares)
* Pilhas e Filas (estáticas e dinâmicas)
* Árvores Binárias (incluindo árvores de busca)
* Tabelas Hash (dispersão)

⚙️ Conceitos importantes:

* Tipos Abstratos de Dados (TADs)
* Organização e manipulação eficiente de dados
* Introdução à análise de complexidade

---

## 🧪 Trabalhos Práticos (TPs)

Os TPs são a parte onde a teoria para de fingir que é fácil.

Aqui são desenvolvidos problemas completos que exigem:

* Implementação do zero
* Organização de código
* Raciocínio lógico
* Atenção absurda a detalhes (porque um ponteiro errado e tudo explode)

📂 Neste repositório você encontrará:

* Resoluções dos TPs
* Estruturas implementadas manualmente
* Exercícios aplicados

---

## 📝 Provas

As provas da disciplina focam principalmente em:

* Entendimento das estruturas
* Aplicação prática dos conceitos
* Lógica de resolução de problemas

💭 Tradução honesta:
não basta decorar, tem que **entender de verdade** como cada estrutura funciona.
